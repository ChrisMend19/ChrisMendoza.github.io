BMP Photo Merger

                  Combine Two BMP Photos into One

Code

In my systems programming class we were tasked with writing a program that takes two BMP photos and blends them together. Also, the user decides a ratio deciding how much of the first photo’s colors are blended with the second photo. Each BMP photo is composed of various RGB pixels. Knowing that I was able to iterate through each pixel and multiply the designated ratio to each of the pixel’s hexadecimal values and write those values to the output file, resulting in a blended photo.

For images of different sizes, I had to take in consideration the different amount of pixels used per image. To handle this, I used bilinear interpolation to map each pixel of the smaller size photo to one or more pixels of the bigger image. This is done by using the distance weighted average of the four nearest pixel values to estimate a new pixel color value.

From this assignment I gained a better understanding of one of the techniques used in image processing. Also, I gained insight on the structure of BMP files and how they can be manipulated to change images.
