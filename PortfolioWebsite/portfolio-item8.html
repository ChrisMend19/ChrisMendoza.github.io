Terminal Arcade

                   an arcade within your command-line shell

Game Code

Inspired by the built-in emulators, emacs, installed using homebrew, I decided to re-create a few of my favorite games in python. Creating a variety of games to played in terminal, I found unique ways of manipulating ASCII characters with and used pyfiglet library to create graphics. Each game implements random package as a way of creating CPUs to play against.

In my hangman game, I manipulated ASCII characters to display the hangman character. After designing the graphics, I used several if statements to determine the outcome based on the user input. python RegEx package handle duplicates and white-spaces in the target word. Once I completed hangman vs cpu, I included two-player functionality to create a similar feel to the orginal hangman.

For the Tic-Tac-Toe I used the dictionary data structure to map each input to the corresponding location. This made it easy to check for 3 in a rows' as I just needed check if a row, column, or diagonal existed on the map bases on there location number. All that was left was some game logic and adding a cpu to play against and tic-tac-toe was made.

In my rendition of blackjack, I used list manipulation to keep track of the 52 cards. This made it efficient to add cards to the user and cpu. A challenge I overcame was handleing different situations with royal cards for the user and the cpu with a few if/else statements. Lastly, I all that was left was to determine the outcome bases on the cards totals and print the winner.
