Minecraft Tile Survival

a version of minecraft that's free

Game Code

Minecraft tile survival is a adventure game where you play as steve and your goal to collect all the diamonds before defeated by the mobs. The game consists of 3 differnt game-modes which increase the speed of the game to make the game more challenging.

The game intially starts with the standard three mobs, however by clicking on the purple portals it triggers a world event to add more functionality to the game. When triggered you can gain the ability to teleport through the portals, at the cost of more dangerous enemies. Mouse Pressed Function also comes in to play when selecting your game mode and whether you'd like to restart the game. Another aspect of code is the enemies utilize A-star pathing algorithm for their tracking, enabling them to trap your character resulting in a defeat. Each enemy is apart of the abstract class 'Entity', which allows for easy creation of enemies, in a similar manner to the Factory Method design pattern.
