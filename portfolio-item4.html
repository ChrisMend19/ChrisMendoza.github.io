Snake Game v.2

                    not your original snake game

Game Code

Snake Game v2 is a java script rendition of the classic snake game. This is my first coding project completed in my intro to computer science course. We utilized the p5.js library to create keyboard movement, random Function, particle systems to bring our game to life. The majority of are character sprites and objects are made of different and vertices. Another key compenent to the game was the translate function, which allowed for movemnet throughout the main game and win screen.
