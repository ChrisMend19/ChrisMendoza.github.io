Flight Fighter

                    an endless side-scroller survival game

Game Code

Flight Fighter, built using unity and C# scripts, is my first mobile survival game. I began this project summer of 2020 and still in the making, as I had to focus more time on college. Using photoshop and unity, I was able to incorporate my 8-bit style sprites and bring them to life in a endless survival android/ios game

I utilized C# scripts to create joystick controls and touch activated shooting for touch screen devices using the Input Function and used random functions to generated a new game each time. Also, I made use of different sprite rendering and sliders to update the health and shoot gauge.

With C# being an object-oriented language, I incorporated a variety of different enemies with unique abilities. Each of these enemies utilize the IEnumerator interface with yield to spawn endless enemies. I created the illusion of a endless background, by translating there location back to the starting location once out of the screen.

Plans for the Game: I hope to finish and publish this game. I want to add more stages to the game with new enemies and different backgrounds. Also, I plan to refactor more creating more hierarchy within each sprite to reduce some of the repitition in the code.
